Introduction

DOM Manipulation refers to the process of using JavaScript to change the content, structure, and style of web pages dynamically. DOM stands for Document Object Model. It allows developers to interact with HTML elements such as headings, paragraphs, buttons, images, and forms.

With DOM Manipulation, developers can create interactive web applications by responding to user actions like button clicks, mouse movements, and keyboard input.

Objectives

  • Understand the concept of DOM.
  • Learn how JavaScript accesses HTML elements.
  • Modify webpage content dynamically.
  • Change CSS styles using JavaScript.
  • Create interactive web pages.

Common DOM Methods

Method Description
getElementById() Selects an element using its ID.
querySelector() Selects the first matching element.
innerHTML Changes the HTML content.
style Changes the CSS style of elements.
addEventListener() Adds events like click or mouseover.
